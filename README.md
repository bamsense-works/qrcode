# Bamsense QR Code Generator

**Open Source. Local. Private.**

Bamsense QR is a privacy-first QR code generator that runs entirely in your
browser. No uploads, no trackers, no server-side rendering. Everything stays on
your device.

## Features
- Live QR generation from text or URLs
- Export to PNG and SVG
- Custom colors, size, margin, and error correction
- Dark mode inspired by the Bamsense design system

## Tech Stack
- React + Vite
- `qrcode` for encoding
- Modern, CSS-driven UI

## Getting Started
```bash
npm install
npm run dev
```
